# YouTube Usage Graph

## Prerequisites
1. MATLAB IDE
2. MATLAB Text Analytics Toolbox <https://de.mathworks.com/products/text-analytics.html>

## Usage
- download Google Takeout Data with YouTube watch history and search history 
- add the absolute or relative path of the subdirectory `/Verlauf/` to `DIR_LIST` of `yt-stats_main.m`
- choose additional parameters if you like
